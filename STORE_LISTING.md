### Long Description (4000 chars max)
```
微简是一款极简生活记录应用，将笔记与记账融为一体。

【为什么选择微简】
• 仅 3 项权限：网络（备份用）、通知（提醒用）、闹钟（提醒用）
• 无存储权限、无位置权限、无相机权限
• 无广告 SDK、无统计追踪 SDK
• 所有数据存储在本地设备，WebDAV 备份用你自己的服务器
• 会员 ¥35 永久买断，无订阅无续费，离线可用

【笔记功能】
• 富文本编辑器：标题(H1-H6)、粗体、斜体、删除线、代码块、引用、列表、待办
• 插入图片、链接、表格、分隔线、附件
• 撤销/重做、字体大小调节
• 分类文件夹、标签、置顶、归档、版本历史
• 单篇笔记锁定、智能筛选
• 导出为 Markdown 或 HTML
• 桌面小组件快速新建便签

【记账功能】
• 快速记录收支，支持多币种
• 智能整理：从备注自动识别金额和分类
• 分类统计、7天收支趋势、每日收支
• 凭证图片附加
• 回收站（误删可恢复）
• 桌面小组件一键记账

【备份与恢复】
• 本地一键导出/导入，保留最近10个历史
• WebDAV 自动云备份到你自己的服务器
• 恢复前预览和安全备份
• 可选密码加密

【隐私与安全】
• 隐私锁（PIN码）
• Xposed/Frida/Hook注入检测
• 会员状态本地加密

【个性化】
• 5种主题风格：经典、Apple、纸张、青植、石墨
• 明暗模式、动态取色(Android 12+)
• 三种字体、三种文字大小
• 中文、英文、法文

最低系统要求：Android 8.0
```
